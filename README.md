# Telegram Bot для списка покупок

Совместный список покупок с возможностью добавления и отметки продуктов.

## Установка

```bash
# Установить uv (если нет)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Установить зависимости
uv sync
```

## Конфигурация

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен
2. Узнайте свой Telegram ID через [@userinfobot](https://t.me/userinfobot)
3. Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_ID=your_telegram_user_id
```

## Запуск

```bash
uv run python bot.py
```

## Команды

### Для всех пользователей

| Команда | Описание |
|---------|----------|
| `/start` | Регистрация / инструкция |
| `/help` | Справка |
| `/add <продукт>` | Добавить продукт |
| `/list` | Показать список |
| `/done <N>` | Отметить купленным |
| `/undo <N>` | Вернуть в список |
| `/remove <N>` | Удалить |
| `/clear` | Удалить все купленные |

### Для администратора

| Команда | Описание |
|---------|----------|
| `/allow @username` | Добавить пользователя в whitelist |
| `/pending` | Список ожидающих подтверждения |
| `/deny @username` | Удалить пользователя |
| `/users` | Список пользователей с доступом |

## Система доступа

1. Новый пользователь пишет `/start` → попадает в список ожидающих
2. Пользователь видит: "Ваш запрос на доступ отправлен администратору. Ожидайте подтверждения."
3. Админ использует `/pending` чтобы увидеть список ожидающих
4. Админ нажимает кнопку **✅ Одобрить** или **❌ Отклонить**
5. Пользователь получает уведомление о доступе

## Кнопки (Reply Keyboard)

Бот предоставляет удобные кнопки для частых действий:

```
┌─────────────────┬─────────────────┐
│  ➕ Добавить    │  📋 Список      │
├─────────────────┼─────────────────┤
│  🗑 Очистить    │  ❓ Помощь      │
└─────────────────┴─────────────────┘
```

## Формат добавления продуктов

**Интерактивный режим (через кнопку ➕ Добавить):**
1. Введите название продукта
2. Выберите единицу измерения (кг, г, шт, л, мл, уп) или "Без меры"
3. Введите количество

**Быстрый режим (через команду):**
```
/add молоко 2л
/add яйца 10 шт
/add сыр 500г
/add хлеб
```

**Объединение дубликатов:** При добавлении одинакового продукта с той же единицей измерения, количество суммируется:
```
/add молоко 2л  →  "молоко 2л"
/add молоко 3л  →  "молоко 5л" (обновлено)
/add молоко 1шт →  "молоко 1шт" (новый продукт, другая мера)
```

## Список покупок

Купленные продукты остаются в списке с отметкой ✅:

```
📋 Список покупок:

1. молоко 2л (добавил: Иван)
2. ✅ хлеб (купил: Мария)
3. яйца 10 шт (добавил: @alex)

👇 Нажмите кнопку под продуктом
```
